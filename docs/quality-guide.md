## 🎨 配色（3色ルール）

**黄金比 70:25:5**

| 役割 | 比率 | 例 |
|------|------|-----|
| ベースカラー | 70% | 白 `#FFFFFF` |
| メインカラー | 25% | 青 `#2196F3` |
| アクセントカラー | 5% | オレンジ `#FF9800` |

### プリセット配色

**解説動画（標準）**
```
ベース: #FFFFFF
メイン: #2196F3
アクセント: #FF9800
```

**エンタメ系（明るい）**
```
ベース: #FFF9C4
メイン: #FF6F00
アクセント: #E91E63
```

### NGパターン

- ❌ 原色そのまま（`#FF0000`）→ 彩度を落とす
- ❌ 5色以上 → 3色以内
- ❌ 背景と文字のコントラスト不足

## 🎬 アニメーション

### 最重要ルール

- ✅ **全てのアニメーションは`useCurrentFrame()`で駆動**
- ❌ **CSSアニメーション禁止**（レンダリングで動かない）
- ❌ **Tailwindアニメーションクラス禁止**

### 推奨設定

| 項目 | 推奨値 |
|------|--------|
| アニメーション時間 | **0.5秒（15フレーム@30fps）** |
| イージング | `Easing.bezier(0.42, 0, 0.58, 1)` |
| spring damping | 10 |
| spring stiffness | 100 |

### コード例

```tsx
// フェードイン
const opacity = interpolate(frame, [0, 15], [0, 1], {
  easing: Easing.bezier(0.42, 0, 0.58, 1),
});

// ポップアップ
const scale = spring({
  frame,
  fps,
  config: { damping: 10, stiffness: 100 },
});
```

## 🔊 音声バランス（-20dBルール）

| 音源 | volume値 | dB |
|------|---------|-----|
| 音声（ナレーション） | 0.8 | -2dB |
| SE（効果音） | 0.5 | -6dB |
| BGM | **0.1** | **-20dB** |

### NGパターン

- ❌ BGM volume 0.5（音声が埋もれる）
- ❌ 音声より大きいBGM
- ❌ フェードなしの開始/終了

## 🎤 VOICEVOX最適パラメータ

```json
{
  "speedScale": 1.0,
  "pitchScale": 0,
  "intonationScale": 1.0,
  "volumeScale": 1.0,
  "prePhonemeLength": 0,
  "postPhonemeLength": 0.1
}
```

### NGパターン

- ❌ speedScale 1.5以上（聞き取りにくい）
- ❌ postPhonemeLength 0（末尾が切れる）
- ❌ prePhonemeLength 0.5以上（前の無音が長すぎ）
